Choose ONE method below and run it in your terminal:

Method 1: Python (Easiest)

Open Command Prompt or Terminal in this folder and run:

python -m http.server 8000

Then open: http://localhost:8000

Method 2: Node.js

npx http-server . -p 8000 -o

Method 3: VS Code Live Server

1. Install "Live Server" extension in VS Code
2. Right-click index.html
3. Click "Open with Live Server"

Method 4: Double Click

Just double-click the start-server.bat file (if on Windows)
